Couleurs de base

Aucune couleur imposée pour l'instant. Ajoutes-en une à quatre : elles seront présentes dans chaque palette générée, les autres viendront les accompagner.

Code couleur

Bouge le point sur l'anneau pour la teinte, dans le triangle pour la saturation et la luminosité. Ou colle un code : hex 3/6 chiffres, rgb(...), trois nombres.
